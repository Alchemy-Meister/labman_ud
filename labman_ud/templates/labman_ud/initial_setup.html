Thanks for installing labman_ud!
...there is still some work to do though ;-)

1. Create a superuser account. This will allow you to get access to the admin site to manage all the model entities labman_ud provides. Having the virtualenv active, execute the following command from the same level as the manage.py file, and follow the instructions on the prompt: 

                                
> python manage.py createsuperuser --settings=labman_ud.settings.local

If you have not initialised a database, run the following command first: 

                                
> python manage.py migrate --settings=labman_ud.settings.local

Now the development server can be re-invoked to run labman_ud locally: 

                                
> python manage.py runserver --settings=labman_ud.settings.local


2. Access the admin site, login with the previous credentials, search for Labman setup > Labman deploy general settings on the left menu, click the add button, fill the form and save the data to end your labman_ud setup.


3. Refresh this website and let the fun begin!
